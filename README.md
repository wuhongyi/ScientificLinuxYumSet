# README

Scientific Linux **yum** 配置文件  
当前目录下有以下文件/文件夹：

- README.md
- sl6
- sl7
- c7

系统版本为 scientific linux 6.x则使用 **sl6**，系统版本为 scientific linux 7.x则使用 **sl7**,系统版本为 CentOS 7,则使用 **c7**.

将文件夹 yum.repos.d 内所有文件复制到 /etc/yum.repos.d 文件夹内

```shell
cd sl[6/7]/c7
cd yum.repos.d
cp   *   /etc/yum.repos.d
# 然后按 y 同意覆盖(提示的每个都需要)
```

然后即可按照教程配置软件。
